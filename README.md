# ApexPlanet Internship - Task 1

## Overview
This project was developed as part of the ApexPlanet Web Development Internship Program (Task 1).

The objective of this task is to learn and apply the foundational concepts of HTML, CSS, and JavaScript by creating a simple interactive webpage.

## Features
- HTML webpage structure
- Headings and paragraphs
- Image integration
- Hyperlinks
- CSS styling (colors, fonts, spacing, and layout)
- Interactive button using JavaScript
- Alert message displayed on button click

## Technologies Used
- HTML5
- CSS3
- JavaScript
